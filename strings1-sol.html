Le stringhe

1. Introduzione 5

Vai avanti per la soluzione

No, "len" tra virgolette verrà interpretato come una stringa, non come una funzione, e quindi Python si lamenterà dicendoci che non possiamo applicare una stringa ad un’altra stringa.

1. Introduzione 6

Domanda 2

Possiamo scrivere una cosa del genere? Cosa produce?

len("len('ciao')")

Vai avanti per la soluzione

Restituisce il numero 11: mettendo il codice Python len('ciao') tra doppie virgolette, è diventato una stringa come tutte le altre. Quindi scrivendo len("len('ciao')") andiamo a contare quanto è lunga la stringa "len('ciao')", cioè 11.

5. Dettagli 1

Le stringhe sono immutabili

Le stringhe sono oggetti immutabili. Tuttavia possiamo:

  • generare una nuova stringa a partire da stringhe precedenti
  • se abbiamo una variabile a cui abbiamo assegnato una stringa, possiamo assegnare un’altra stringa a quella variabile

Nell'esempio sulla destra generiamo una nuova stringa a partire da altre definite precedentemenete

L'operazione + quando eseguita tra stringhe le concatena creando una NUOVA stringa.

5. Dettagli 2

Riassegnare variabili

Altre variazioni allo stato di memoria possono essere ottenute riassegnando le variabili

Se una stringa che è stata creata non risulta assegnata a nessuna variabile, Python si occuperà automaticamente di eliminarla dalla memoria.

Cosa è successo?

x = 'ciao'
x = x + 'mondo'

  2. Nella seconda linea Python ha:
    • calcolato il risultato di x + 'mondo', generando la NUOVA stringa 'ciaomondo'
    • assegnato la stringa appena generata alla variabile x

6. Formattazione 1

Formattare stringhe

Concatenare stringhe con un segno + come sopra è macchinoso e si presta ad errori.

Ci sono diverse soluzioni migliori, per una recensione completa ci riferiamo al sito Real Python. In particolare, guardare le più utili che sono le f-strings, disponibili per Python >= 3.6

6. Formattazione 2

Formattare con %

  • Questa soluzione non è la migliore, ma è molto usata e supportata in tutte le versioni di Python, perciò l’abbiamo adottata nel libro
  • Abbiamo usato un place-holder (segnaposto) %s dentro la stringa, per dire a Python di sostituirlo con una variabile
  • Per passare a Python la variabile, dopo la stringa abbiamo messo un simbolo % seguito da una variabile

6. Formattazione 3

Formattare con %

  • Se volessimo usare più variabili?
  • Bisogna solo aggiungere più segnaposti %s, e dopo il % mettere la lista di variabili in delle parentesi tonde
