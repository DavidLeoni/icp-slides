What are booleans?

  • Used in boolean algebra
  • The type is bool
  • The truth values are either True or False

Conversions

  • We have seen the possible values for booleans are True and False
  • However, booleans can be also converted to integers, and vice versa!
  • How are the truth values handled when converting integers to booleans? Play with the code on the right, and try to guess!
  • Every integer is evaluated as True, except for 0
  • True behaves like a 1
  • False behaves like a 0

Evaluation order

  • If during the evaluation of a boolean expression Python finds out that there can only be one possible result, the next expressions are not even evaluated
  • For instance, when evaluating False and x, Python doesn't need to check x's value, since the first False allows us to understand the final result of the expression can only be False

Evaluation order (2)

  • Instead, if the evaluation continues, its final result will be the result of the final expression

Evaluation order (3)

  • What about the or operator?
  • The idea is the same - if the first expression already resolves to true, there is no need to continue the evaluation
