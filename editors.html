Introduction

The following web components are exposed by the library

  • <{language}-editor />
  • <{language}-editor-input />
  • <{language}-editor-dark />
  • <{language}-editor-input-dark />
  • <{language}-editor-v />
  • <{language}-editor-v-input />
  • <{language}-editor-v-dark />
  • <{language}-editor-v-input-dark />
  • <{language}-editor-tabs-dark />

The languages currently available are: Javascript, Typescript, Java, C++

<{language}-editor-dark />

<{language}-editor-input-dark />

Language: Javascript
