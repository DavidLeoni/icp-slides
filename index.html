Slides Customization Web Application

Some Use Cases

  • A professor prepares the material for a programming laboratory in which students are encouraged to run code and play with it
  • A professor wants to give to his students a common yet easy-to-use environment to code
  • A student wants to take interactive notes during a programming laboratory
  • A student doesn't have enough space in his computer to download a complete IDE
  • Someone wants to create a slide deck that supports people with vision Impairments

Main Requirements

  • Slides must be downloadable and accessible through any web browser
  • When possible, the compilation and run processes should happen client-side
  • Compilation and run time should be limited to a maximum of 5 seconds
  • The Web Application should at least provide tools to add a title, a paragraph, images and runnable code
  • During the development a set of example slide decks should be created

Technologies

  • HTML slides generation using Reveal.js
  • No external service for running browser-native programming languages
  • External API to compile and run the other programming languages
  • Use of WebAssembly when possible, to limit the access to our API
  • CodeMirror v6 open-source in-browser code editor
