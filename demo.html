Showcase of the Interactive Code Playgrounds project

Learning to code isn't easy.
Your university probably teaches programming through a lab course.
The first step of the lab, very often, is to make students install a coding environment.
This is quite complex. And without the coding environment, you can't start coding.
But it doesn't need to be.
Let's start by rethinking one of the staples of programming classes.
The slide deck.

And this is all front-end code! Which means this works even if you are offline (encourage to try). Or, more realistically, this means you can integrate ICPs in pretty much any LMS or platform without installing complex components.
