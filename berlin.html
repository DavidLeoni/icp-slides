The typical programming lab slide
  • Code embedded in slides as images/plain text
  • Hard (or impossible) to copy/paste
  • As teacher, changes are hard to make
  • Not accessible for e.g., screen readers
Most importantly: checking code results isn't easy!

Credits to geeksforgeeks for the code sample

Interactive Code Playgrounds
  • Run code directly on slide
  • Easy to modify/fix mistakes
  • No need to install anything
