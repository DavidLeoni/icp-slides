The project

What is it? The idea is to provide HTML Slides with runnable code

Help Students Learning Programming Languages

Support People with Vision Impairments

  • ARIA Attributes
  • Semantic tags
  • Images' alt
  • Clear DOM Structure

Used Technologies

  • Reveal.js for the HTML slides generation
  • CodeMirror as embedded code editor
  • WebAssembly to compile and run code without external servers (when possible...)
  • External API for other programming languages that can't be directly compiled in the browser

ICP-bundle

The current solution

Some completely customized components have been created and exported via a single JS script

Exported web components and functions can be embedded in any web page by simply importing this script

<script src='icp.bundle.js'></script>
